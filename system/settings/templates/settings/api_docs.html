UESO-PMIS API Documentation

A guide to integrating with UESO's Project Monitoring System. Scalable, secure, and RESTful for university-wide connectivity.

Steps to Access

Create Account

Begin by registering a standard user account on the UESO-PMIS portal. This establishes your identity within the university system.

Access Settings

Navigate to your User Profile dashboard. Look for the "Developer Settings" tab to manage your external connections.

Request Key

Select the API Tier that matches your needs and submit a request. Our admins will review your application shortly.

Connect with your system

Once approved, use your generated key to authenticate your external applications.

Features

System Capabilities

Full Analytics

GET

/analytics/api/all-project-data/

Retrieve a comprehensive snapshot of the extension office. This read-only endpoint nests events, documents, and evaluation scores into a single JSON response.

Event Management

CRUD

/api/calendar/meeting-events/

A flexible calendar interface. Create new meeting events, invite participants, and manage schedules. Users can strictly edit only events they own.

Expense Tracking

POST

/api/projects/{id}/expenses/

Specialized for Project Leaders. Upload receipts, log costs, and track budget utilization in real-time against the approved proposal.

Data Aggregation

GET

/archives/api/aggregate/

Optimized for dashboards. Returns high-level statistics, project counts by status, and financial summaries without the heavy data load.

Technical Stuff
